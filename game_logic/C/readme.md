## третье
сортировка красно-черным деревом как и тимсорт имеет сложность 
в среднем O n*log(n) 
в худшем случае O n*log(n) 
и в лучшем случае O n
сложность обусловлена вставкой n элементов в дерево сложность каждой вставки O log(n)
